ELECROW Display Setup Tutorial
===================================
This is a guide and tutorial for setting up an ELECROW ESP32 7.0 Inch HMI Display (DIS08070H) [[specs]](https://mm.digikey.com/Volume0/opasdata/d220001/medias/docus/5791/ESP32-7.0-Specification.pdf). All files referenced throughout this tutorial are available in this same repository and are free to use. It is important to note that there are 3 types of files used in the tutorial:  
- **Tutorial Project Assets:** Files used for creating the UI presented in this tutorial, including the entire platformIO project and UI assets, use if you desire to follow along or wish to have a reference vantage point.  
  
- **Tutorial Page Assets:** Images used to illustrate the tutorial better, ignore.
    
- **Common Assets:** Files provided for general use in various projects, highly recommend using these as they help tidy up your future Elecrow Display shenanigans. (Jump to section # for more details about this files.)
